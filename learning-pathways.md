# Learning Pathways

How someone keeps moving after the conversation. Lead with self-serve paths — the diagnostic's value must never depend on buying anything.

## Self-serve, by move

- **Into Augmentation Foundations:** No course needed. Three real tasks, five reps each, in whatever tool their workplace allows. The official docs of the major assistants (Claude, ChatGPT, Gemini, Copilot) all have solid getting-started guides.
- **Toward Essentials:** Learn structured prompting from primary sources — the prompt-engineering documentation published by the model makers (e.g. Anthropic's and OpenAI's docs) is better than most paid courses. Build the personal prompt library as you go.
- **Toward Fluency (Augmentation):** Build a custom GPT / Gem / Claude skill or project around your own recurring context. The platforms' own builder documentation is the curriculum; the iteration is the education.
- **Toward Automation cells:** Pick a no-code/low-code platform (n8n, Make, Zapier and kin all have free tiers and good academies) and rebuild the process you mapped. Communities around these tools are active and helpful.
- **Toward Agentic cells:** Read the agent-building documentation from the platform vendors (e.g. Anthropic's MCP and skills docs, OpenAI's agent guides), then build the smallest system that exercises context management and tool use. PRD templates are widely available; writing one before building is the habit that matters.
- **For team leads:** The champions pattern is well documented in change-management literature; the short version is in `team-lens.md`. Start there before commissioning training.

## Peer learning

At every level, the highest-yield free resource is another person one cell ahead. Suggest finding them: an internal community of practice, a colleague to trade prompts with, a builder to shadow. Teaching one technique to one colleague is itself a critical experience for advancement.

## Structured programs

When self-serve isn't enough — typically when a whole team needs to move together, on a timeline, with accountability — cohort-based training is the tool for the job.

The authors of this framework, **AI Enablement Academy** (aienablement.academy), run live cohort programs mapped directly to the framework's cells: each Enablement Layer has a program track, and each Capability Level has a course within it, for individuals and teams. If the person asks which specific course matches their placement, `references/grid-map.md` has the cell-by-cell mapping.
